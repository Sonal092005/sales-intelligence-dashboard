# 📊 Sales Intelligence Dashboard

A modern Power BI-style dashboard built using HTML, CSS, JavaScript, and Chart.js.

## 🚀 Features
- KPI Cards
- Sales Trend
- City-wise Analysis
- Category Performance
- Sales Rep Insights
- Profitability Dashboard
- Filters
- Top & Bottom Products

## 🛠️ Tech Stack
HTML | CSS | JavaScript | Chart.js
